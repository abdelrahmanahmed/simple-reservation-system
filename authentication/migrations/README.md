# Knex migrations

## Install
```
npm i
```

## Add migration and seed
```
npm run add-migration ${MIGRATION_NAME}
npm run add-seed ${SEED_NAME}
```

## Run migration and seed
```
npm run run-migrations
npm run run-seed
```

## Rollback migration
```
npm run rollback-migration
```
